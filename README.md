# Data Digger - SQL Project

## Project Overview
Data Digger is a practical SQL project that provides hands-on experience in managing a MySQL database for an E-Commerce Store.

## Database Schema
The project includes four main tables:
- Customers
- Orders
- Products
- OrderDetails

## Queries Performed
- CRUD operations (Create, Read, Update, Delete)
- Aggregate functions (MAX, MIN, AVG, SUM, COUNT)
- Filtering and sorting operations
- Foreign key relationships

## Assumptions Made
1. Used AUTO_INCREMENT for primary keys
2. Set appropriate data types for each field
3. Established proper foreign key relationships
4. Used DECIMAL(10,2) for monetary values

## How to Run
1. Execute the SQL script in MySQL
2. Run individual queries as needed
